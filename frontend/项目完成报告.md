## 🚀 快速开始

### 1. 环境要求
- 现代浏览器（Chrome 90+、Firefox 88+、Safari 14+）
- HTTPS环境（用于文件上传功能）
- 后端API服务器运行

### 2. 部署步骤
```bash
# 1. 复制前端文件到您的项目目录
cp -r frontend/* /path/to/your/project/app/static/

# 2. 确保后端API服务正在运行
# 3. 访问应用
# https://your-domain.com/
```

### 3. 配置说明
所有API端点都在 `frontend/js/main.js` 中配置，可以根据您的后端API进行调整：

```javascript
const config = {
    api: {
        baseUrl: 'https://your-backend-domain.com',
        timeout: 10000
    },
    upload: {
        maxFileSize: 5 * 1024 * 1024, // 5MB
        allowedTypes: ['image/jpeg', 'image/png', 'image/webp']
    }
};
```
